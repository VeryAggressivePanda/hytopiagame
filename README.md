## Pushing to GitHub

Gebruik SSH voor deze repo:

```bash
git push git@github.com:VeryAggressivePanda/hytopiagame.git main
# of geforceerd indien nodig:
git push -f git@github.com:VeryAggressivePanda/hytopiagame.git main
```

Zorg dat je SSH key is toegevoegd aan je GitHub-account (`ssh -T git@github.com` moet “Hi …” tonen).
